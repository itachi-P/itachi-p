    - 目指すゴールの方向性
      - 設計・マネジメント・教職（ブリッジSE・メンター等）＞　外国人とのチームの一員　＞　自ら手を動かす受託開発（個人事業）＞ 1人でなんでもできるス～パ～ハカー
